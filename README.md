# COMP2152 Study Quiz

[![Live Demo](https://img.shields.io/badge/Live_Demo-brightgreen?style=for-the-badge)](https://Luistello87.github.io/COMP2152-Quiz-App/)

Welcome to the **COMP2152 Study Quiz**! This interactive browser-based web application is designed to help you prepare for exams by testing your knowledge across a variety of fundamental Python concepts, Tools, Git, and software engineering principles.

With zero setup required, you can click the Live Demo button above to immediately engage with the quiz engine, test your knowledge, shuffle questions, and get instant feedback.

## Features

- **Interactive UI**: A clean, responsive design utilizing Tailwind CSS.
- **Topic Selection**: Choose to test your knowledge on specific individual subjects or study all of them simultaneously.
- **Randomizer**: Mix up the question order on demand using the shuffle toggle.
- **Answer Checking**: Receive instant feedback on correct and incorrect selections.
- **Score Results**: See your final score and a visual breakdown of your performance by topic.
- **Review Mode**: Review all selected answers to learn from mistakes after the quiz ends.

## Study Topics
Over **72 questions** are organized across the following core lecture materials:

- **Control Flow** — if/elif/else logic (4 questions)
- **Python Lists** — indexing & negative indexing (5 questions)
- **Python Slicing** — inclusive/exclusive rules (3 questions)
- **Python Tuples** — immutability, speed, syntax (4 questions)
- **Python Sets** — operations, mutability, uniqueness (6 questions)
- **Dictionaries** — keys, values, methods (5 questions)
- **Git** — add, commit, push, branch, fork, clone, fetch (9 questions)
- **Debugging** — breakpoints, Step Over/Into/Out (4 questions)
- **OOP** — polymorphism, inheritance, encapsulation, decorators (9 questions)
- **Recursion & Algorithms** — operation logic, base cases (3 questions)
- **Functions & Modules** — def keyword, logic imports (3 questions)
- **File I/O** — text, basic operations, CSVs (3 questions)
- **System Applications** — exception handling, platform, sys, os (3 questions)
- **Databases** — SQLite connections, queries, committing (3 questions)
- **Classes & Objects** — object definitions, structures, instantiating (3 questions)
- **Open Source Licenses** — BSD, MIT, GNU, Fair License (5 questions)

## Local Usage
Since this web application relies solely on client-side React processing through Babel standalone, it requires absolutely no build steps. 

1. Simply clone the repository or download `index.html`.
2. Double-click the file to run it in any modern web browser.
